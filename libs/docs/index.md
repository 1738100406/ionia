# 快速上手

## 环境

首先得有 node，并确保 node 版本是 v12.16.1 或以上。

```shell
$ node -v
v12.16.1
```

## 启动项目

```shell
$ git clone git@github.com:ioniajs/ionia.git --depth=1 && cd ionia

$ yarn && yarn start
```

访问 http://localhost:7000/

## 启动组件库

```shell
$ yarn libs:dev
```

## 启动文档

```shell
$ yarn docs:dev
```

访问 http://localhost:8000/
